# MANUAL DE USUARIO

Bienvenido al sistema de punto de venta. Este manual te explica de manera simple cómo buscar un producto, registrar una venta y consultar el corte de caja.

## 1. Buscar un producto

1. Abre el sistema e inicia sesión si te lo pide.
2. Busca el campo de búsqueda de productos. Suele estar en la pantalla principal o en la sección de ventas.
3. Escribe el nombre del producto o una parte del nombre.
4. Presiona la tecla `Enter` o haz clic en el botón de buscar.
5. Revisa los resultados que aparecen. El sistema debe mostrar el producto con su precio y cantidad disponible.
6. Si no encuentras el producto, intenta escribir otra palabra o revisa si el nombre está bien escrito.

## 2. Registrar una venta

1. Selecciona el producto que quieres vender.
2. Verifica el precio y la cantidad del producto.
3. Ingresa la cantidad que el cliente desea comprar.
4. Añade el producto a la venta. Esto puede ser un botón que dice “Agregar”, “Añadir” o similar.
5. Repite los pasos 1-4 si el cliente compra más de un producto.
6. Cuando tengas todos los productos, revisa el total de la venta.
7. Elige el método de pago: efectivo, tarjeta u otro que ofrezca el sistema.
8. Completa la operación con el botón de “Cobrar”, “Finalizar venta” o similar.
9. Si el cliente paga en efectivo, ingresa el dinero entregado y el sistema mostrará el cambio.
10. Guarda o imprime el recibo si el sistema ofrece esa opción.

## 3. Consultar el corte de caja

1. Ve a la sección de corte de caja o cierre de jornada. Puede llamarse “Corte”, “Cierre de caja” o “Caja”.
2. Selecciona la fecha o el turno correspondiente si el sistema pide esa información.
3. Revisa los totales que muestra el sistema: ventas totales, efectivo, tarjeta y otros pagos.
4. Comprueba que el monto final coincida con el dinero físico en la caja.
5. Si necesitas, guarda un reporte o haz una captura de pantalla para tu registro.
6. Si el sistema permite cerrar la caja, presiona el botón de “Cerrar”, “Finalizar corte” o similar.
7. Confirma la operación cuando el sistema te lo pida.

## Consejos prácticos

- Usa siempre el nombre correcto del producto para encontrarlo rápido.
- Verifica precios y cantidades antes de finalizar la venta.
- Revisa el corte de caja al final del turno para detectar errores.
- Si tienes dudas, pide ayuda a un supervisor o a la persona responsable del sistema.

¡Listo! Con estos pasos puedes usar el sistema para buscar productos, registrar ventas y consultar el corte de caja de forma sencilla.
